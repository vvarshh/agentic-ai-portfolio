# LifeOS Agent
Status: Building

Problem:
Busy professionals waste mental energy planning their day.

Solution:
A multi-tool agent connected to Telegram and Google Sheets that automates planning and summarizes every day at 6pm.

Current Features:
- /todo + /buy + /note task capture
- Auto-save to Sheets
- Daily summary at 6pm

Next Features:
- Calendar integration
- Weekly summary agent
